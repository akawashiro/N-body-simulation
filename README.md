# Toy N-body Simulation

This is a toy n-body simulator written in C++. You can play with the following commands.

```
git clone https://github.com/akawashiro/N-body-simulation.git
cd N-body-simulation
mkdir -p build
cd build
cmake ..
make
./main
```
